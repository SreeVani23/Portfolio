Dashboard for digital market data using PowerBI

The dataset consists of features such as Response (target), Complain, Date of customer's enrollment, Customer Education, Marital status, the number of small kids in the customer's household, the number of teenagers in the customer's household, Customer Income, Amount spent on fish, meat, fruits, sweets, wines, and gold products in the last 2 years, the number of purchases made with a discount, the number of purchases made using a catalog, the number of purchases made directly in stores, the number of purchases made through the company's web services, the number of visits to the company's website, and the number of days since the last purchase. Using these features, we created a dashboard based on key performance indicators.

The dashboard explores:

    1) Income generated with respect to the country.
    2) How a person's income is affecting their purchasing habits.
    3) How the presence of kids is affecting a family's buying habits.
    4) Whether web visits are transforming into actual purchases.
